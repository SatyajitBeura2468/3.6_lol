How to use The Dynamics

Run controlled experiments and connect each visual change to the continuity equation and Bernoulli equation.

  1. Increase flow rate. Particle speed and dynamic pressure rise everywhere.
  2. Reduce the throat diameter ratio. The same discharge must pass through less area, so throat velocity rises sharply.
  3. Watch the static pressure field. Faster sections generally show lower static pressure.
  4. Turn on head loss to include a simplified viscous energy loss based on Reynolds number and hydraulic diameter.
  5. Drag vertically near the constriction to reshape it directly. Hover anywhere inside the pipe to inspect local values.

Keyboard: Space pauses, R resets, and H opens this guide.
